Project Title: PC Builder Cloud Application.

Description:
This initiative is a web application that runs in the cloud and enables the user to create their own PCs, compatibility, save builds, and order using the Stripe payment facility.

Technologies Used:
- Python
- Django
- JavaScript
- Stripe API

Dependencies:
Install using:
pip install -r requirements.txt

Environment Variables (.env):
DJANGO_SECRET_KEY=your_secret_key
DEBUG=True
STRIPE_SECRET_KEY=your_stripe_key
STRIPE_PUBLISHABLE_KEY=your_publishable_key

Run Locally:
python manage.py migrate
python manage.py runserver

Features:
- User registration and log in.
- PC assembler with compatibility tests.
- Save builds
- Checkout and payment through Stripe.
- Admin dashboard
